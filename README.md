# GitHub Groups

[ignition-devs](https://github.com/ignition-devs) - An organization fostering script development in Ignition.

# Tutorials

[Git Ignition Lab](https://github.com/ia-eknorr/git-ignition-lab) - Introduction to Git-based version control in Ignition

# Tools

[Ignition Lint](https://github.com/ia-eknorr/ignition-lint) - Linter for Ignition views and scripts

[Ignition Flint](https://github.com/keith-gamble/ignition-flint) - VS Code extension that enriches the Ignition development experience

[Kindling](https://github.com/inductiveautomation/kindling) - A standalone collection of utilities to help Ignition users ([forum post](https://forum.inductiveautomation.com/t/kindling-a-collection-of-tools-to-help-with-ignition/54689))

Ignition API ([8.1](https://github.com/ignition-devs/ignition-api-8.1), [8.3](https://github.com/ignition-devs/ignition-api-8.3)) - A Python package that allows developers to get code completion for Ignition Scripting API scripting functions in their IDE of choice.

[Incendium](https://github.com/ignition-devs/incendium) - A collection of utility project scripts to extend the core `system` library.
